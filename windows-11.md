# Windows 11

## Uninstall programs

Download Bulk Crap Uninstaller, it's an open source program to properly uninstall programs.

## Manage startup program

To precisely select which program will run automatically at login, download : [Sysinternals Suite](https://learn.microsoft.com/en-us/sysinternals/downloads/sysinternals-suite) and run `autoruns64`

## Disable telemetry and unwanted feature

Download and run [O&O ShutUp10++](https://www.oo-software.com/en/shutup10/download)
